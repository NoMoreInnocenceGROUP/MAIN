# Documents

This directory contains all working materials used throughout the evaluation and review process. It serves as the primary workspace where information is collected, analyzed, refined, and either finalized or archived.

The structure supports a clear lifecycle: **active analysis → completion → archival**.

---

## Directory Structure

### `Current/`

This section contains:

* Newly identified threats or suspicious activity
* Ongoing evaluations that are not yet complete
* Uncertain or evolving assessments
* Draft analyses requiring further review or validation

Content in this folder is considered **active work-in-progress** and may change frequently as new information becomes available.

---

### `Finalized/`

This section contains:

* Completed evaluations and full reviews
* Confirmed classifications and structured conclusions
* Personal assessment ratings based on available evidence
* Fully resolved analysis cases

Entries here represent **completed analytical work** that has been reviewed and concluded.

---

### `Archived/`

This section contains:

* Older documents no longer actively referenced
* Superseded or outdated evaluations
* Abandoned or incomplete analyses deemed no longer relevant
* Historical records kept for reference purposes

Content in this folder is maintained for **traceability and historical context**, but is not actively used in ongoing evaluations.

---

## Workflow Overview

1. New cases are added to `Current/`
2. Analysis and review are performed until a conclusion is reached
3. Completed evaluations are moved to `Finalized/`
4. Outdated or irrelevant entries are transferred to `Archived/`
