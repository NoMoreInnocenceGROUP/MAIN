# Data

This directory contains finalized evaluation outputs produced through structured analysis and review processes. Each entry represents a completed assessment that has passed through documentation, evaluation, and conclusion stages within the workflow.

The contents are organized by severity and complexity of the assessed material.

---

## Directory Structure

### `Generic/`

This section contains finalized evaluations related to:

* Low to medium risk threats
* Common or obvious scam patterns
* Newly emerging but straightforward malicious methods
* Rapid assessments requiring fast classification and response

Evaluations in this category prioritize **speed, clarity, and practical handling guidance** over deep technical breakdowns.

Typical outputs include:

* Scam breakdown summaries
* Pattern identification reports
* Quick-risk classification notes
* Basic mitigation recommendations

---

### `Threats/`

This section contains finalized evaluations related to:

* High-risk or complex threats
* Phishing infrastructure and malicious websites
* Coordinated or multi-step scam operations
* Technically detailed or behaviorally sophisticated attack patterns

Evaluations in this category emphasize:

* Deep technical analysis
* Behavioral and structural breakdown
* Operational pattern mapping
* High-confidence conclusions supported by evidence

These entries are designed for **precision, thoroughness, and detailed understanding of threat mechanisms**.

---

## Classification Principle

All entries are categorized based on:

* Severity of the threat
* Complexity of the system or method involved
* Required depth of analysis
* Urgency and clarity of the conclusion
